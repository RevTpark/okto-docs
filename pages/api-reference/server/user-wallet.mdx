Fetch all the wallets created by a user using their `user_id`.

<ColumnLayout>
    <>
        ## Headers
        `x-api-key` : string <br />
        The S2S API Key provided by Okto.

        ---
        
        ## Query Parameters
        `user_id` : string <br />
        The user id of the user whose portfolio data is to be fetched.

        ---
    </>
    <>
        ## Code
        ```bash copy
        curl -X GET '${baseUrl}/s2s/api/v1/wallet?user_id=80f5b553-4650-4943-9721-8db41d78ccc9'
        -H 'accept: application/json' 
        -H 'x-api-key: API_KEY'
        ```
