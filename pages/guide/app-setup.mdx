Before we start exploring Okto, we will use some packages and tools to help us in the development process.

<Steps>
    ### Install required packages
    First lets install the required packages for navigation and screens. We will use `@react-navigation/native`, you can check the installation guide [here](https://reactnavigation.org/docs/getting-started).
    <Tabs items={['pnpm', 'npm', 'yarn']}>
        <Tabs.Tab>
            ```bash filename="bash" copy
            pnpm i @react-navigation/native react-native-screens react-native-safe-area-context
            ```
        </Tabs.Tab>
        <Tabs.Tab>
            ```bash filename="bash" copy
            npm i @react-navigation/native react-native-screens react-native-safe-area-context
            ```
        </Tabs.Tab>
        <Tabs.Tab>
            ```bash filename="bash" copy
            yarn add @react-navigation/native react-native-screens react-native-safe-area-context
            ```
        </Tabs.Tab>
    </Tabs>

    ### Setup Navigation
    We will first create a file with the name `navigation.tsx` in the root of the project and add create a stack navigator as shown below.

    ```tsx filename="navigation.tsx" copy
    import { createNativeStackNavigator } from 'react-native-screens/native-stack';

    const Stack = createNativeStackNavigator();

    export default function Navigation() {
        return (
            <Stack.Navigator>
                // Add screens here
            </Stack.Navigator>
        );
    }
    ```

    Now we will import the `Navigation` component in `App.tsx` and wrap it with `NavigationContainer` from `@react-navigation/native`.
    ```tsx filename="App.tsx" copy
    import { NavigationContainer } from '@react-navigation/native';
    import Navigation from './navigation';

    ...

    export default function App() {
        return (
            <NavigationContainer>
                <Navigation />
            </NavigationContainer>
        );
    }
    ```

    ### Setup Screens
    Now we will create a dummy screen to test the navigation. Add a new folder with the name `screens` in the root of the project and create a file with the name `HomeScreen.tsx` in the root of the project.
    
    ```tsx filename="HomeScreen.tsx" copy
    import { View, Text } from 'react-native';

    const HomeScreen = () => {
        return (
            <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
                <Text>Home Screen</Text>
            </View>
        );
    }

    export default HomeScreen;
    ```

    Now we will import `HomeScreen` component in `navigation.tsx` and add it to the stack navigator.
    ```tsx filename="navigation.tsx" copy
    import HomeScreen from './screens/HomeScreen';

    export default function Navigation(){
        return (
            <Stack.Navigator initialRouteName="Home">
                <Stack.Screen name="Home" component={HomeScreen} />
            </Stack.Navigator>
        )
    }
    ```

    Now can use `navigation.navigate("Home")` to navigate to the `HomeScreen` from any other screen.
