### Initialize the SDK
You need to first intialize the SDK with the API key to get started. 
You can do this in the root file of your app, for example `App.tsx` or `index.tsx`.
```javascript filename="App.tsx" copy
import { init } from "rn-okto-sdk";

...

const apiKey = 'YOUR_API_KEY';
init(apiKey);
```

And we are doneðŸŽ‰! You are now ready to use the SDK in your app.
You can optionally setup the authentication feature of the SDK by following the next step.

### Intialize Authentication (Optional)
If you want to use the authentication feature of the SDK, you need to initialize it with the `clientId` provided by Google Console Platform.
The method additonally returns array of all the wallets created on successful authentication.
```javascript filename="App.tsx" copy
import { authenticate } from "rn-okto-sdk";

...

const clientId = 'YOUR_CLIENT_ID';
authenticate(clientId, (result, error) => {
  if (error) {
    // Handle errror
    console.error(error);
  } else {
    // Returns an array of all the wallets created
    const wallet = JSON.parse(result);
    console.log(wallet);
  }
});
```
