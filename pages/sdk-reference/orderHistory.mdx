
# orderHistory Method
This method is used to get all the order details of the current user.

## Parameters
- `callback` - Callback function to be called after API is executed. This function contains two parameters:
  - `result` - Orders if API is successful.
  - `error` - Error message if any.

```tsx copy
orderHistory(
    callback: (result: any, error: any) => void
) => void;
```

## Example
```tsx copy
import { orderHistory } from 'rn-okto-sdk';

interface Order {
    order_id: string
    network_name: string
    order_type: string
    status: string
    transaction_hash: string
}

orderHistory((result, error) => {
      if(result){
          const orderHistory: Order[] = JSON.parse(result);
          console.log(orderHistory);
      }
});
```
