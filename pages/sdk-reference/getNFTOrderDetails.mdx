
# getNFTOrderDetails Method
This method is used to get all the NFT order details of the current user.

## Parameters
- `callback` - Callback function to be called after API is executed. This function contains two parameters:
  - `result` - NftOrder if API is successful.
  - `error` - Error message if any.

```tsx copy
getNFTOrderDetails(
    callback: (result: any, error: any) => void
) => void;
```

## Example
```tsx copy
import { getNFTOrderDetails } from 'rn-okto-sdk';

interface NftOrder {
  order_id: string;
  user_id: string;
  status: string;
  network_name: string;
  entity_type: string;
  collection_address: string;
  nft_id: string;
  order_type: string;
  tx_hash: string;
}

getNFTOrderDetails((result, error) => {
      if(result){
          const nftHistory: NftOrder[] = JSON.parse(result);
          console.log(nftHistory);
      }
});
```
