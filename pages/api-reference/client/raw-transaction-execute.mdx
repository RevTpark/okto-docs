Execute a raw transaction on a network. The API returns a `jobId` which can be used to track the status of the transaction.

<ColumnLayout>
    <>
        ## Parameters
        `network_name` : string <br />
        Name of network to execute transaction on.

        ---
        `transaction` : object <br />
        Transaction object.

        ---
        `transaction.from` : string <br />
        Address of sender.

        ---
        `transaction.to` : string <br />
        Address of receiver.
        
        ---
        `transaction.data` : string <br />
        Data to be sent with transaction.

        ---
        `transaction.value` : string <br />
        Value to be sent with transaction.

        ---

    </>
    <>
        ## Code
        ```bash copy
        curl -X POST '${baseUrl}/api/v1/rawtransaction/execute' 
        -H 'accept: application/json' 
        -H 'Authorization: Bearer AUTH_TOKEN' -d '
            {
                "network_name": "POLYGON_TESTNET",
                "transaction": {
                    "from": "0x0342A54DD44E8744FD185579Af57845Cb0ac6cB0",
                    "to": "0x80322ea18633A1f713e987d65Ae78AcCDAB6e55e",
                    "data": "0x",
                    "value": "0x10000"
                }
            }
        '
        ```

        ## Response
        ```json
        {
            "status": "success",
            "data": {
                "jobId": "d6ce7f0c-1a7d-441d-bb3e-4e431fae84c9"
            }
        }
        ```
    </>
</ColumnLayout>
