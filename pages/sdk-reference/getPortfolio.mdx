
# getPortfolio Method
This method is used to get all the token assets in the user's portfolio.

## Parameters
- `callback` - Callback function to be called after API is executed. This function contains two parameters:
  - `result` - Tokens if API is successful.
  - `error` - Error message if any.

```tsx copy
getPortfolio(
    callback: (result: any, error: any) => void
) => void;
```

## Example
```tsx copy
import { getPortfolio } from 'rn-okto-sdk';

export interface Portfolio {
  token_name: string;
  token_image: string;
  quantity: string;
  amount_in_inr: string;
}

getPortfolio((result, error) => {
    if(result){
        const tokens: Portfolio[] = JSON.parse(result);
        console.log(result);
    }
});
```
